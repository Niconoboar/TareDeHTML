La etiqueta h

La etiqueta "h" se utiliza para definir los encabezados en HTML. La diferencia entre "h1", "h2", "h3", "h4", "h5" y "h6" radica en su nivel de importancia y tamaño de fuente, siendo "h1" el más importante y grande, y "h6" el menos importante y más pequeño.

Las etiquetas h1, h2, h3, h4, h5 y h6 no solo indican el tamaño de texto, también representan diferentes niveles de encabezado en una página web. Cada etiqueta de encabezado (h1 a h6) tiene un tamaño de fuente predeterminado, donde h1 es el más grande y h6 es el más pequeño. Además del tamaño, los encabezados transmiten información sobre la estructura y jerarquía del contenido de la página. Los encabezados más grandes, como h1 y h2, se utilizan para títulos principales y secciones importantes, mientras que los encabezados más pequeños, como h3 a h6, se usan para títulos secundarios o subdivisiones de contenido.

< h1 >

h1 sirve para identificar el titulo principal de la pagina web

< h2 >

h2 presentar un nivel de encabezado secundario en una página web

< h3 >

h3 se utiliza para representar un nivel de encabezado terciario en una página web.

< h4 >

h4 se utiliza para representar un nivel de encabezado cuaternario en una página web

< h5 >

h5 indica un nivel de encabezado quinario en una página web

< h6 >

h6 indica un nivel de encabezado de menor jerarquía en una página web, siendo el de menor importancia entre todos los niveles de encabezado.

< p >

La etiqueta p en HTML se utiliza para agrupar y formatear el contenido de texto en párrafos separados dentro de una página web. Al usar esta etiqueta, se crea un bloque de texto independiente que facilita la legibilidad y estructura del contenido. Es especialmente útil cuando se necesita presentar información en forma de párrafos distintos, como en artículos, descripciones o secciones de texto en general..

Al decir que la etiqueta p sirve para formatear, nos referimos a que permite aplicar un estilo visual predefinido al texto contenido dentro de ella. Esto incluye la separación automática del texto en párrafos separados, con un espaciado adecuado entre ellos. Además, el formato por defecto puede incluir márgenes, alineación y ajuste del texto, lo que contribuye a una presentación más legible y estructurada del contenido en la página web.

< span >

La etiqueta span en HTML se utiliza para modificar partes específicas de texto dentro de un párrafo o una sección sin afectar al resto del contenido. Puedes usarla para aplicar estilos o realizar cambios visuales en palabras o frases concretas.

La etiqueta < span > por sí misma no altera visualmente el texto en la página web. Es un elemento en línea que se utiliza para aplicar estilos o manipulaciones específicas a partes seleccionadas de texto. Para que el texto se vea alterado, es necesario aplicar estilos CSS o atributos a la etiqueta ``. Puedes agregar reglas de estilo CSS, como cambiar el color, el tamaño de fuente o aplicar efectos visuales, para que el texto dentro de la etiqueta `` se vea modificado.

< ! -- comentarios -- >

La secuencia < ! -- -- > se utiliza en HTML para crear comentarios. Todo lo que se encuentre dentro de estos delimitadores no será visible en la página web, pero se puede utilizar para incluir notas, explicaciones o instrucciones para otros desarrolladores que revisen el código. Los comentarios son útiles para documentar el código y realizar anotaciones sin afectar el resultado visual o funcional de la página.

Hipervinculo

La etiqueta "a" en HTML se utiliza para crear enlaces o hipervínculos en una página web. Permite a los usuarios hacer clic en el texto o la imagen envuelta dentro de la etiqueta "a" para acceder a otra página web, un archivo descargable u otra ubicación dentro de la misma página. Se utiliza el atributo "href" para especificar la dirección o URL a la que se debe dirigir el enlace.
En este caso voy a crear un hipervinculo con google

La etiqueta < a target="_blank" href="http://www.google.ec" > crea un enlace o hipervínculo en una página web. Al hacer clic en este enlace, se abrirá una nueva pestaña del navegador con la página web especificada en el atributo href. Esto permite al usuario mantener la página actual abierta mientras accede al enlace de destino. En este caso, el enlace dirige al sitio web de Google en el dominio "http://www.google.ec".Enlace a Google en nueva pestaña

< img / >

El código img src="img/imagen.png"/ se utiliza para insertar una imagen en una página web. En este caso, se especifica la ruta de la imagen "imagen1.png" dentro de la carpeta "img". Hay que tener en cuenta que la estructura de carpetas y el nombre de la imagen sean correctos y que la imagen exista en esa ubicación relativa. (Aquí la imagen se encuentra en su tamaño original).



La expresión "img/" se refiere a una carpeta llamada "img" en el directorio actual o relativo del archivo HTML en el que se encuentra el código. Cuando se especifica src="img/imagen1.png", se está indicando que la imagen "imagen1.png" se encuentra dentro de esa carpeta llamada "img".
Es recomendable manejar las imagenes denrto de una carpeta, por lo que se creó la carpeta img dentro de la carpena que usamos para guardar todo lo relacionado con nuestra pagina web de html.

En este caso, la imagen se mostrará con un ancho de 400 píxeles, lo que puede afectar su tamaño visual en la página web.

Formularios

Apertura del formulario y campo nombre

< form action="/formulario" method="post"> < label for="nombre">Nombre
< input id="nombre" name="nombre" placeholder="Ingrese su nombre aquí">

La etiqueta < form action="/formulario" method="post"> representa la apertura de un formulario en HTML. El atributo action especifica la URL o ruta donde se enviarán los datos del formulario, y el atributo method especifica el método de envío, en este caso, el método POST, en este caso, el elemento de etiqueta < label> y el elemento de entrada de texto < input>. Dentro de la etiqueta < form>, definimos los elementos que permiten a los usuarios ingresar y enviar datos.
< label for="nombre">Nombre< />: Esta etiqueta < label> se utiliza para asociar un texto descriptivo con un elemento de formulario, en este caso, un campo de entrada de texto. El atributo for enlaza la etiqueta con el elemento de formulario correspondiente a través de su valor, que en este caso es "nombre". De esta manera, al hacer clic en la etiqueta "Nombre", se activará el campo de entrada correspondiente.

< input id="nombre" name="nombre" placeholder="Ingrese su nombre aquí" >: Este elemento < input> crea un campo de entrada de texto donde los usuarios pueden ingresar su nombre. El atributo id establece un identificador único para este campo de entrada, que coincide con el valor del atributo for de la etiqueta < label>. Esto asegura la asociación correcta entre la etiqueta y el campo de entrada. El atributo name especifica el nombre del campo que se enviará al enviar el formulario. El atributo placeholder muestra un texto de ejemplo dentro del campo de entrada como sugerencia para los usuarios.



Botón

< input type="submit" id="Botón" value="Aplastar Botón">
< label for="Botón" >

Estamos creando un botón en el formulario. Para personalizar el texto que se muestra en el botón, hemos utilizado el atributo value en la etiqueta < input>. Al asignarle el valor "Aplastar botón" al atributo value, el texto "Aplastar botón" será el que se mostrará en el botón.
Por otro lado, la etiqueta < label> se utiliza para asociar un texto descriptivo con un elemento de formulario. En este caso, el atributo for de la etiqueta < label> está establecido en "Botón", lo que indica que está asociado al elemento < input> que tiene el id "Botón". Esto mejora la accesibilidad del formulario.

Botónes

con el objetivo de entender de mejor manera con la que funcionan los botones agregaremos un formulario muy corto en los tipos de boton. La etiqueta button es mucho mas personalisable.

botón de tipo botón

< button type="button">Botón

es un botón genérico en HTML que no tiene una acción asociada por defecto. Esto significa que cuando se hace clic en el botón, no ocurrirá ninguna acción a menos que se le agregue un evento JavaScript o se asocie a una función
< button>: Es la etiqueta que crea el botón.
type="button": El atributo type especifica el tipo de botón, en este caso, "button" indica que es un botón de acción normal.
"Botón": Es el texto o contenido del botón que se muestra en la interfaz.


botón de tipo reset

< button type="reset">Borrar

El código representa un botón de reinicio en HTML.
Cuando se hace clic en este botón, se restablecerán los valores de los campos de formulario a sus valores predeterminados. Esto significa que si el formulario tiene campos de entrada de texto, casillas de verificación seleccionadas o botones de opción marcados, al hacer clic en el botón de reinicio, todos esos campos volverán a su estado inicial.
< button>: Es la etiqueta que crea el botón.
type="reset": El atributo type especifica el tipo de botón, en este caso, "reset" indica que es un botón de reinicio.
"Borrar": Es el texto o contenido del botón que se muestra en la interfaz.


botón de tipo enviar

< button type="submit">enviar

Cuando presionemos el botón los datos ingresados en los campos de formulario se enviarán al servidor para su procesamiento. El comportamiento exacto del envío del formulario dependerá de cómo se haya configurado el servidor y cómo se haya diseñado el manejo de los datos del formulario.
Es importante tener en cuenta que el botón de envío solo tendrá efecto dentro de un formulario. Debe colocarse dentro de las etiquetas de apertura y cierre del formulario para que sea parte de ese formulario en particular.
< button>: Es la etiqueta que crea el botón.
type="submit": El atributo type especifica el tipo de botón, en este caso, "submit" indica que es un botón de envío.
"enviar": Es el texto o contenido del botón que se muestra en la interfaz.
(El atributo action del formulario especifica la URL a la que se enviarán los datos del formulario. Ya que este solo es un ejemplo dicha URL no está configurada correctamente en el servidor, lo que provoca un error al intentar enviar el formulario.)
